# READMORE

READMORE is the public documentation and contribution entry point for Random Knights docs that are safe to share outside the internal READLESS workspace.

## Folder Map

- `architecture/aieds/` - AI Environmental Disclosure Standard drafts and supporting public architecture notes.
- `architecture/earth/` - Earth feature, source, attribution, and governance specs that have been sanitized for public review.
- `architecture/ecosystem/` - public product and app-shell architecture notes.
- `architecture/packages/` - public package boundary and scaffold notes.
- `automation/` - public automation and BDD guidance.
- `branding/` - public brand, theme, and icon documentation.
- `decisions/` - public architecture decision records.
- `kn1ghts/`, `orac1es/`, `rand0m/`, `uti1ity/`, `_c1assr00m/` - public app and experience notes.

Internal operations, deployment details, security posture, environment variables, agent identities, roadmap state, and historical records remain in `READLESS`.
